# cloud-vfs skill — Azure Blob storage for ML repos

Companion skill for the [cloud-vfs](https://github.com/sahasrarjn/cloud-vfs) package.

Install into a project:

```bash
pip install git+https://github.com/sahasrarjn/cloud-vfs.git
cloud-vfs init --skill
```

This copies this skill to `.cursor/skills/azure-blob-storage/` so Cursor agents discover it automatically.

See the main README: https://github.com/sahasrarjn/cloud-vfs
